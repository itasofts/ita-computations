2. App Description 

The App is designed for land surveyors, Geomatics engineers and students in these fields for computing Bearing and Distance, Area computations, preparing Beacon Index, and Plan Data sheets. The "Enter Your Coordinates Here" menu allows user to enter coordinates in the format (Northings, Eastings, Label (N, E, L)). coordinates can either be added one at a time by adding a row to the GUI, or adding multiple rows an inputing the coordinates manually.The next option is the copy and paste single or multiple rows in to the "Paste Coordinates X,Y,Beacon" text box and clicking on the "Paste & Add Rows" button and the copied coordinates will be added to the GUI.There is an option to import CSV files saved in the format suggested by clicking on the Import CSV (N, E, L) button to add the coordinates to the GUI. User can then click on the "Save and Calculate" button for the calculations to be done.The App dynamically calculates and creates the aforementioned sheets, which can be verified and printed or saved to PDF.
